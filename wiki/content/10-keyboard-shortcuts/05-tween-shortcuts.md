## Tween Shortcuts

| Shortcut | Tween Type |
| --- | --- |
| `Alt+P` | Position |
| `Alt+S` | Scale |
| `Alt+O` | Opacity |
| `Alt+R` | Rotation (Z-axis) |
| `Alt+Y` | Rotation (Y-axis) |
| `Alt+C` | Crop |
| `Alt+B` | Blur |
| `Alt+V` | Volume |
| `Alt+H` | Hue |
