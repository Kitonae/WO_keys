## Timeline Controls

| Shortcut | Action |
| --- | --- |
| `Spacebar` | Play/Pause toggle |
| `Numpad 0` | Start playback |
| `Escape` | Pause |
| `Numpad *` | Jump to last start |
| `Numpad +` | Zoom in timeline |
| `Numpad -` | Zoom out timeline |
| `Ctrl+I` | Insert layer |
| `Ctrl+Delete` | Delete layer |
| `Ctrl+P` | Add play cue |
| `Ctrl+Shift+P` | Add pause cue |
