## Main Window Overview

The WATCHOUT interface is divided into several dockable windows that can be arranged to suit your workflow.

### Menu Bar

The menu bar at the top provides access to all commands organized in logical groups: File, Edit, Stage, Media, Timeline, Tween, Window, and Help.

### Toolbar

Quick access buttons for common operations like play, pause, and zoom controls.

### Window Areas

The main workspace is divided into panels that can be rearranged, resized, or floated as separate windows.
