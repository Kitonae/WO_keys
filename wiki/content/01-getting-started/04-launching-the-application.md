## Launching the Application

After installation, launch WATCHOUT from the Start menu or desktop shortcut.

### First Launch

On first launch, you'll see the **Welcome screen** with options to:

- **Create New Show** – Start a fresh project

- **Open Recent** – Access recently edited shows

- **Open Show** – Browse for an existing .watch file

### The Main Interface

The WATCHOUT interface consists of several key windows:

- **Stage** – Visual preview of your displays and content

- **Timeline** – Where you arrange and time your media

- **Assets** – Your media library

- **Properties** – Settings for selected items

- **Network** – Connected display servers
