## Stage Navigation

| Shortcut | Action |
| --- | --- |
| `Ctrl+Shift+D` | Frame all displays |
| `Ctrl+Shift+O` | Scroll to origin |
| `Ctrl+1` to `5` | Set zoom level |
| `Ctrl+Alt+Drag` | Pan view |
| `Ctrl+Wheel` | Zoom in/out |
| `Ctrl+Arrow` | Nudge selection 1px |
| `Ctrl+Shift+Arrow` | Nudge selection 10px |
