## Opening Existing Shows

### Open Methods

- **File → Open** or `Ctrl+O` – Browse for a show file

- **Recent Shows** – Access from File menu

- **Double-click** – Open .watch files directly from Windows Explorer

### Connecting to a Director

To join an existing show session on the network, use **File → Connect to Director** and select the active show.
