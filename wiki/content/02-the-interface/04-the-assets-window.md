## The Assets Window

The Assets window is your media library – all images, videos, and audio files available for use in your show.

### Adding Media

- Drag files directly into the window

- Use **Media → Add Media File**

- Right-click and select **Add Media**

### Organization

Create folders to organize your assets. Right-click and select **Add Folder**.

### Asset Information

Select an asset to see its properties: resolution, duration, file path, and codec information.
