## Adding Displays

### Adding a Standard Display

- Go to **Stage → Add Display**

- A new display appears on the Stage

- Position it to match your physical screen arrangement

- Set the resolution in the Properties panel

### Display Properties

- **Name** – Descriptive label for the display

- **Width/Height** – Resolution in pixels

- **Position** – X, Y coordinates on stage

- **Output** – Which display server output to use
